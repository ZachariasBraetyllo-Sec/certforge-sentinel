# Cybersecurity Certification Guide

Synthetic knowledge source for CertForge Sentinel.

## Purpose

This guide defines example cybersecurity certification paths for learners in technical roles. It is synthetic and created for hackathon demonstration only.

1. Google Cybersecurity Certificate
2. Security+
3. SC-900

Recommended focus areas:

- Security fundamentals
- Network security basics
- Linux and SQL basics
- SIEM alert review
- Incident response process
- Authentication and access control
- Common threat types
- Security documentation

Suggested learning sequence:

1. Security foundations
2. Networking basics
3. Operating systems and command line
4. Detection and monitoring
5. Incident response
6. Identity and access management
7. Practice questions and readiness review

## General Readiness Guidance

A learner is more likely to be ready for a certification attempt when they:

- Complete the recommended core topics
- Score at least 75% on practice assessments
- Can explain weak areas in their own words
- Complete at least 15 to 25 focused study hours
- Review missed questions and repeat weak topics
- Can apply concepts to realistic scenarios
